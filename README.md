#  React Portfolio Website

This is a professional portfolio website built with **React.js** and **Vite**. The project is fully containerized using **Docker** for consistent and easy deployment.

---

##  Prerequisites
Before running the project, make sure you have the following installed:
* [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Running)

---

##  How to Run the Project (The Easy Way)

The fastest way to get the project up and running is using **Docker Compose**. 

1. **Open your terminal** in the project's root directory.
2. **Run the following command:**
   ```bash
   docker-compose up -d
